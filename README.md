# My Site - [Check it out](https://tomascorridoni.com/)

Porfolio site to showcase some of my work.

The design of the site it's self is very simple but it has quite a few custom interactive elements made to give it personality and highlight information.

## Development & Deployment

This is an old Vue CLI 3 / webpack 4 project — it only builds on **Node 14 or 16**
(Node 17+ breaks on the OpenSSL 3 change). The pinned version lives in `.nvmrc`.

```bash
nvm use            # picks up 14.18.0 from .nvmrc
npm install --legacy-peer-deps
npm run serve      # local dev
npm run build      # build to dist/
npm run deploy     # publish dist/ to the gh-pages branch (custom domain via public/CNAME)
```

Hosted on GitHub Pages from the `gh-pages` branch, served at the custom domain
`tomascorridoni.com`.
